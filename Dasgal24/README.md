## Бодлогын тайлбар (Монгол хэлээр)

Өгөгдсөн хоёр хүүхэдтэй бинари модны (binary tree) зангилаануудыг түвшин бүрээр нь (level order) зүүнээс баруун тийш хэвлэх.

- Өндөрөөс доош, зүүнээс баруун тийш аялах аргачлал.
